
# របៀប Import Profile នៅក្នុង Visual Studio Code (VS Code)

## ១. ទាញយកឯកសារ Profile
- ប្រើឯកសារ `Chantha.code-profile` ដែលមាននៅក្នុង folder នេះ។

## ២. បើកកម្មវិធី VS Code

## ៣. ចូលទៅកាន់ Import Profile
1. ចុចលើរូបអាយខន "Account" នៅខាងក្រោមឬខាងឆ្វេង (សញ្ញាមនុស្ស)
2. ជ្រើស "Profiles: Import Profile..."
3. ជ្រើសប្រភេទឯកសារ (zip/json/code-profile) ហើយចុច "Continue"
4. រើសឯកសារ `Chantha.code-profile`
5. ចុច "Import"

## ៤. ចាំរង់ចាំអោយ VS Code Import រួច
- បន្ទាប់ពី Import រួច VS Code នឹងបង្ហាញសារ "Profile imported successfully"
- អ្នកអាចប្រើប្រាស់ Settings, Extensions, Keybindings, ... ដូចក្នុង profile នោះបានភ្លាមៗ
