# របៀបដំឡើង Visual Studio Code (VS Code)

## ១. ទាញយក (Download)
- ចូលទៅកាន់គេហទំព័រ [https://code.visualstudio.com/](https://code.visualstudio.com/)
- ចុចប៊ូតុង "Download for Windows" ឬ "Download for macOS" អាស្រ័យលើប្រព័ន្ធប្រតិបត្តិការ (OS) របស់អ្នក។

## ២. ដំឡើងលើ Windows
1. បន្ទាប់ពីទាញយករួច ចុចដើម្បីបើកឯកសារ (.exe)
2. ចុច "Next" បន្តៗ
3. ទទួលយក License Agreement
4. ជ្រើសទីតាំងដំឡើង (អាចទុកដូចដើម)
5. ចុច "Install" ហើយរង់ចាំអោយដំឡើងរួច
6. ចុច "Finish" ដើម្បីបញ្ចប់

## ៣. ដំឡើងលើ macOS
1. បន្ទាប់ពីទាញយករួច បើកឯកសារ (.zip ឬ .dmg)
2. ទាញ VS Code ទៅក្នុង "Applications"
3. បើកកម្មវិធី Visual Studio Code ពី Applications

## ៤. បើកប្រើប្រាស់ VS Code
- បន្ទាប់ពីដំឡើងរួច អ្នកអាចបើក VS Code ដោយចុចលើអាយខន (icon) របស់វា។
- អ្នកអាចដំឡើង Extensions ដើម្បីជួយអោយការសរសេរកូដកាន់តែងាយស្រួល។
